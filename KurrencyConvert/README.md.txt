# KurrencyConvert - Convertisseur de devises

## 📌 Objectif du projet
L'application **KurrencyConvert** permet de convertir des montants entre différentes devises en temps réel. Elle offre une interface intuitive pour sélectionner les devises source et cible, saisir un montant, et obtenir instantanément le résultat de la conversion.

## ⚙️ Technologies utilisées
- **Android Studio** (SDK 35) avec Kotlin
- **Jetpack Compose** pour l'UI moderne
- **Retrofit 2 + Gson** pour les API REST
- **Coroutines** pour le traitement asynchrone
- **Firebase Realtime Database** pour la persistance
- **Firebase Analytics** pour le suivi d'utilisation
- **Material Design 3** pour le système de design
